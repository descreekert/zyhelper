## 已知 Bug / 待修

1. **暗黑模式 CSS 不完整** — Tailwind utility class（bg-white、text-slate-900 等）在 dark mode 没有 dark: 变体覆盖
2. **schoolTag 复合字符串处理** — primaryTier() 提取了主标签做徽章颜色，但筛选 chip 用 includes 匹配，**有遗漏的复合标签可能漏选**（如 "保研资格" 不在 chip 选项里）
3. **enumerations.provinces 含重复** — meta.json 里 "江苏"/"江苏省"、"浙江"/"浙江省" 都存在，没去重
4. **enumerations.cityTiers 含复合值** — 如 "一线城市/省会城市"、"三线城市/省会城市"，chip 选中 "一线城市" 不会命中 "一线城市/省会城市"
5. **冲稳保按钮覆盖筛选条件无提示 / 无撤销** — 用户点完不知道筛选被改了
6. **筛选 chip "其他"** — 当前 schoolTags 没有 "其他" 这个值，硬编码的 chip 选不到任何东西
7. **筛选 chip 包含奇怪值** — schoolTags 里有 "保研资格"、"乡村振兴人才培养优质校" 等，但我硬编码的 chip 只展示 985/211/双一流/国重点/省重点/其他
