## 详细实施

### Item 1 — ≡ 移到筛选 panel 内
- 顶栏不再有 `≡` 按钮
- `FilterPanel` 顶部加 `‹` 收起按钮 (跟收起态的 `›` 一致)
- 一致性: 收起 ← 在 panel 内; 展开 → 浮在主内容左侧

### Item 2 — 浮按钮位置
- 改 `top-[100px]` → `top-[140px]`
- 改 `left-2` → `left-0` (贴边)
- 改 `w-6 h-12` → `w-5 h-10` (更小)
- 现在不遮挡 "N 条结果" (它在 main 顶部 ~140px 之上)

### Item 3 — 默认不含中外合作
- `initialFilters.includeMidOutside`: `true` → `false`
- 默认筛后从 700 → 229 条 (排除掉中外合作的)

### Item 4 — 关键词去 label
- 删除 `<label>🔍 关键词</label>`, 留输入框 (placeholder 已含 "🔍 学校 / 专业类 / 专业")

### Item 5 — header 重排
- 标题 "📚 志愿助手" (去掉 "填报")
- 顶栏第 1 行: 标题 + ScoreTool (26 分数/位次/等位换算) + 右侧操作按钮
- 顶栏第 2 行: 冲稳保 3 按钮独占一行 + 数据生成信息
- `score-tool` 组件不再包含冲稳保按钮, 移到 header 第 2 行 inline 调用 `onApplyTier`

### Item 6 — 刷新移到表格 head
- 从顶栏移到表格 head bar (与 ⚙ 列 / 每页 同行)
- 改 `reloadData()`:
  - 不再调用 `load()` (set loading=true)
  - 改为 `Promise.all` 静默 fetch 4 个 JSON, cache-bust
  - 失败 alert; 成功 console.log
  - filters/columns/sortKeys/expanded 等所有 state 全部保留

### Item 7 — 展开行布局大重构
**当前问题**:
- 学校/专业 两块高度不一致 (右边 信息少, 左边 信息多, 视觉不协调)
- 预测/变化 两块如果没变化, 右半边空白
- 25vs26 表的"专业名"列宽太窄, 多行换行, 行高乱

**新布局**:

### Item 8 — 单展开模式
```js
function toggleExpand(id) {
  const wasOpen = store.expandedRows.has(id);
  store.expandedRows = wasOpen ? new Set() : new Set([id]);
}
```
- 点击其他行 → 旧的自动关闭, 新的展开
- 点击当前展开行 → 关闭
